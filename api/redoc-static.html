Go to item-table to learn more about VERA data structures.

get single object

get data for a task

path Parameters
task_id
required
string
Example: M0139m

id of the task

Responses

Response Schema: application/json
id
required
string^[A-Za-z_]+$

Identifier for task

label
required
string

Short internal name. Used in Lists and for filenames.

title
required
string

Use this title in booklets.

subject
required
string
Enum: "V3MA" "V3DE" "V8MA" "V8DE" "V8EN" "V8FR"

Refers to VERA-Projekt (VERA3 or VERA8) and to subject (mathematics, German, English, French)

Array of objects (Metadata Entry)

List of metadata

itemMetricsReliability
string
Enum: "FULL" "LIMITED" "NONE" "NOT_AVAILABLE"

Important if the items have metrics (frequency of solution, logits etc.) and these metrics are not fully reliable.

yearOfTryout
integer
yearOfPreviousApplication
integer
yearOfApplication
integer
